yascroll.el
===========

Overview
--------

yascroll.el is Yet Another Scroll Bar Mode for GNU Emacs.

Screenshots
-----------

![](http://cx4a.org/software/yascroll/yascroll.png)

Usage
-----

To toggle yascroll for all buffers, `M-x global-yascroll-bar-mode`.
